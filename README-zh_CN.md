## URL Decode Online 是什么？

**URL Decode Online** 是一个为开发者打造的轻量级在线工具，用于对 URL 编码字符串进行解码与编码，适用于处理查询参数、API 请求、重定向链接以及各种被转义的 URL 数据。

你可以使用它来：

* 将 URL 编码字符串还原为原始文本  
* 将普通文本编码为可安全用于 URL 的格式  
* 正确处理 UTF-8 和多字节字符  
* 自动解析多层 URL 编码（例如 `%2520` → `%20` → 空格）  

所有操作都在浏览器本地完成，不会上传任何数据。

---

## 为什么要做这个工具？

目前很多在线 URL 工具存在以下问题：

- 页面广告多、体验混乱  
- 对 UTF-8 与中文支持不完整  
- 多层编码解析不正确  
- 不适合开发者日常调试  

**URL Decode Online** 的目标是：

> 提供一个简洁、快速、可靠、对开发者友好的 URL 编码与解码工具。
